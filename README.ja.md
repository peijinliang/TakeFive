## コンセプト

リマインダーに一日を管理させるのではなく、TakeFive が時間を静かに覚えておきます。ルールを一度設定し、次の通知を確認したら、あとは作業に集中できます。

### 開発者の毎日に合う小さな習慣

| シーン | 今できること | これから検討する方向 |
| --- | --- | --- |
| **目を休めるリズム** | 45 分または 60 分ごとの整列リマインダーと短い休憩アクション。 | 20-20-20 プリセットと半透明/全画面の休憩画面。 |
| **立つ・伸ばす・水分補給** | 水分、ストレッチ、カスタム習慣を定期通知。会議中は一時停止。 | 30 秒、1 分、3 分の動作カード。 |
| **睡眠の境界線** | 仕事を終えるためのワンショット通知。 | 就寝カウントダウンと「今夜は遅くまでコードを書かない」モード。 |
| **カフェインの確認** | データをローカルに保存し、簡単な確認ルールを作成。 | カフェイン記録と睡眠への影響の目安。 |
| **いたわり付きポモドーロ** | 間隔、静かな時間、手動の休憩セッションを組み合わせる。 | 集中の終わりに水分やストレッチを提案。 |
| **軽いセルフチェック** | 「20 秒遠くを見る」などの低負荷な通知。 | 医療判断ではない目の疲れセルフケアカード。 |
| **仕事の振り返り** | リマインダーイベントと結果をローカルに保存。 | 集中時間、夜更かし、休憩、水分の週間レビュー。 |
| **静かなデスクトップ相棒** | トレイ常駐、オフライン動作、メインウィンドウを閉じても継続。 | オフライン呼吸練習、ホワイトノイズ、終業前カード。 |

右側は製品の方向性であり、すべてが現在のリリースに含まれるわけではありません。

## TakeFive の特徴

- **控えめな通知。** 右下のフローティングサーフェスはフォーカスを奪いません。初期設定では 7 秒後に閉じ、完了・スヌーズ・スキップ・閉じるを選べます。
- **バックグラウンドで正確に動作。** 権威ある時刻計算は Rust スケジューラーが担当し、ブラウザーやフロントエンドの `setTimeout` に依存しません。
- **スリープや再起動に対応。** 起動、スリープ復帰、ロック解除、時刻・タイムゾーン変更後も SQLite から再照合し、古い通知をまとめて表示しません。
- **プライバシー優先。** アカウント、クラウド、テレメトリー、キーボード入力、画面、ウィンドウタイトル、マイク、カメラ、マウス軌跡を必要としません。
- **理由がわかる。** 各ルールの次回時刻を表示し、一時停止や静かな時間、表示されなかった理由を説明します。

## 現在の機能

| 領域 | MVP に含まれる機能 |
| --- | --- |
| ルール | 固定時刻、アンカー基準の間隔、ワンショット、平日/毎日、適用時間帯、昼休み除外 |
| 管理 | 作成、概要と次回時刻の確認、有効化、無効化、論理削除 |
| 通知 | 右下の透明サーフェス、7 秒（設定可能）の自動クローズ、完了/スヌーズ/スキップ/閉じる、キュー、システム通知へのフォールバック |
| 常駐 | トレイ、単一インスタンス、メインウィンドウを閉じても継続、Windows 起動時の自動起動（初期オン、変更可能） |
| 一時停止 | 30 分、1 時間、2 時間の全体一時停止。毎日 12:00–13:30 の静かな時間（深夜跨ぎ対応） |
| 言語 | 英語、簡体字中国語、日本語、スペイン語の UI |
